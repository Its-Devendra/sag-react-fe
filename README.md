# React Web App (Codigo Frontend)

## Setup Instructions

Ensure that npm is installed. Can check using this command:

```
npm --version
```

1. Install dependencies.
   ```
   npm i
   ```
2. Install husky `pre-commit` and `pre-push` hooks.
   ```
   npx husky install
   ```
3. Run the app.
   ```
   npm run start
   ```
